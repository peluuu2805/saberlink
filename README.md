# SaberLink ⚔️

Control BLE de sable láser estilo Star Wars.

**Funciones:**
- Encender / apagar el sable por Bluetooth
- Selección de color del cristal Kyber
- 4 perfiles de sonido (Jedi, Sith, Clone Wars, Mandaloriano)

**Uso:** Abre la app en Chrome Android y conecta al dispositivo `SableLaser`.

*Requiere Web Bluetooth — Chrome en Android*
